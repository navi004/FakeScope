# 🔍 FakeScope — Fake Engagement Detection in Social Media

> **OrgX Hackathon — Problem Statement 3**  
> Detecting artificial social media engagement using real behavioural data and machine learning.

---

## 📌 Problem Statement

Social media platforms are flooded with bot accounts that generate fake engagement, distorting genuine behavioural patterns, misleading advertisers, and manipulating public discourse. This project builds a behavioural detection model that:

- Differentiates **organic vs. artificial engagement**
- Detects **coordinated behavioural anomalies**
- Produces an **Authenticity Score** and **Bot Probability** per account
- Explains **which behavioural signals** drove each prediction

## 🔧 Feature Engineering

All **33 features** are engineered behavioural signals derived from raw account data, grouped by the 4 PS indicators:

### ⏱️ Timing Regularity (5 features)
| Feature | Behavioural Meaning |
|---|---|
| `tweets_per_day` | Posting velocity — bots post 50–200+/day |
| `account_age_days` | Account lifetime — very new + active = bot signal |
| `tweet_density` | Total tweets ÷ account age — high = robotic |
| `timing_regularity_score` | tweets_per_day ÷ log(age) — bots score extremely high |
| `lifetime_tweet_rate` | Tweets per hour of account's life — bots approach 24/7 |

### 💥 Engagement Burst Patterns (9 features)
| Feature | Behavioural Meaning |
|---|---|
| `total_tweets` | Absolute tweet count |
| `total_likes_given` | Engagement given — bots give near-zero |
| `likes_per_day` | Daily engagement rate |
| `is_high_tweeter` | Binary: >50 tweets/day |
| `abnormal_tweet_rate` | Binary: >100 tweets/day |
| `burst_index` | Total tweets ÷ age — sudden large volume |
| `burst_flag` | Binary: burst_index > 20 |
| `sudden_fame` | New account (<30 days) with >1000 followers |
| `engagement_paradox` | Many followers but zero likes given |

### 🌐 Network Interaction Patterns (6 features)
| Feature | Behavioural Meaning |
|---|---|
| `followers_count` | Total followers |
| `friends_count` | Total following |
| `follower_friend_ratio` | Followers ÷ Following — extremes = bots |
| `friends_to_followers` | Following ÷ Followers — high = follow-spam |
| `likes_to_tweets` | Engagement reciprocity ratio |
| `network_anomaly_flag` | Binary: follows >2000 but tiny follower ratio |

### 🗣️ Linguistic Consistency (13 features)
| Feature | Behavioural Meaning |
|---|---|
| `has_description` | Has bio — bots often skip |
| `description_length` | Bio character length |
| `description_word_count` | Bio word count — richer = more human |
| `has_location` | Location set — human signal |
| `has_default_pic` | Default avatar — strong bot signal |
| `is_default_profile` | Default theme — bot signal |
| `is_verified` | Verified account — strong human signal |
| `has_geo` | Geo-enabled — human behaviour |
| `profile_completeness` | Sum of 5 setup signals (0–5) |
| `screen_name_length` | Username length |
| `screen_name_digit_ratio` | Digit fraction in username — bots like user_48291847 |
| `bio_has_url` | Bio contains URL — spam signal |
| `bio_is_generic` | Bio <15 chars — templated/unfilled |

## 🤖 Model

**Algorithm:** Random Forest Classifier (scikit-learn)

**Why Random Forest:**
- Excellent on tabular behavioral data
- Naturally handles mixed feature types (binary flags + continuous)
- Built-in feature importance for behavioral explainability
- Robust to outliers (bots create extreme feature values)
- No scaling required — ideal for this feature set

**Class Imbalance Handling:** Balanced class weights (`class_weight='balanced'`)

**Output per account:**
- `Bot Probability` (0–100%)
- `Authenticity Score` = 100 − Bot Probability
- `Behavioural Flags` — specific anomalies detected

## 💡 Key Behavioural Insights

1. **Tweet volume** is the single strongest bot signal — bots post at superhuman rates
2. **Profile incompleteness** (no bio, default picture, default theme) is a reliable bot cluster
3. **Engagement paradox** — accounts with many followers but zero likes given are almost always bots
4. **Follower/following imbalance** reveals coordinated follow-back schemes
5. **Verified + old accounts** are the strongest human signals in the dataset
6. **Sudden fame** (new account, many followers) flags purchased or manufactured audiences
